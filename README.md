## Webpack-React-Quizapp
This is the webpack version of the react app [**QuizApp-Frontend**](https://github.com/nforesperance/QuizApp-Frontend.git)
## Running the app
**Installing**

    npm install
**Running in development**

    npm run dev

**Building**

    npm run build
